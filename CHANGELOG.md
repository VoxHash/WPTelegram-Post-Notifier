## [1.0.0] - 2024-01-01

### Added
- Complete Telegram API integration with rate limiting and error handling
- Advanced templating system with dynamic token system and multiple parse modes (MarkdownV2, HTML, Markdown)
- Smart routing rules for category, tag, and post type-based routing to different destinations
- Async processing using Action Scheduler for non-blocking notification queue
- Comprehensive logging system with filtering, search, and export capabilities
- Health monitoring integration with WordPress Site Health
- Modern React-based admin interface with real-time testing and preview
- Gutenberg sidebar integration for post-level notification controls
- Complete REST API for external integrations and testing
- Secure token storage with encryption and capability management
- WooCommerce support for product notifications and price changes
- Bulk actions for sending notifications to multiple posts
- Template preview functionality with different parse modes
- Rate limiting with exponential backoff for API calls
- Comprehensive error handling and retry logic
- Full internationalization support with POT template
