# Example 01 — Basic Setup

Step-by-step guide to setting up WP Telegram Post Notifier.

## Prerequisites

- WordPress 6.3+
- PHP 7.4+
- Telegram account
- Bot token from @BotFather

## Steps

1. **Get Bot Token**
   - Open Telegram and search for @BotFather
   - Send `/newbot` and follow instructions
   - Save the bot token

2. **Add Bot to Channel**
   - Create or use existing Telegram channel
   - Add bot as admin
   - Note channel ID

3. **Configure Plugin**
   - Go to Settings > Telegram Notifier
   - Enter bot token
   - Add destination channel
   - Test connection

4. **Customize Template**
   - Go to Template tab
   - Customize message format
   - Preview and save

5. **Test**
   - Create a test post
   - Publish it
   - Verify notification in Telegram
