# Usage

Common commands and workflows.

## Basic Usage

Once configured, the plugin automatically sends notifications when:
- New posts are published
- Posts are updated (if enabled)
- Scheduled posts are published

## Advanced Usage

### Custom Templates

Use dynamic tokens in your message templates:

```
New post: {post_title}

{post_excerpt}

Read more: {post_url}
```

### Routing Rules

Set up different destinations for different content:
- Category-based routing
- Tag-based routing
- Post type routing

### Bulk Operations

Send notifications to multiple posts at once using bulk actions.
