# FAQ

Frequently asked questions.

## Can I send to multiple channels?

Yes! You can add multiple destinations in the plugin settings. Each destination can be a different channel, group, or private chat.

## Can I customize the message format?

Absolutely! The plugin includes a powerful template system with dynamic tokens. You can customize the message format in the Template tab.

## Does it work with WooCommerce?

Yes! The plugin has special support for WooCommerce products, including price change notifications and product-specific tokens.

## Can I send notifications manually?

Yes! You can send notifications manually using the "Send to Telegram" action in post lists or the Gutenberg sidebar panel.

## Is it safe to use?

Yes! The plugin follows WordPress security best practices, including input sanitization, capability checks, and secure token storage.
